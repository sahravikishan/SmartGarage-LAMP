# SmartGarage — Car Service History Tracker

Full-stack LAMP application deployed on AWS EC2.

## Tech Stack
- Ubuntu 24.04 LTS
- Apache 2.4
- MySQL 8.x  
- PHP 8.3
- Glassmorphism UI

## Features
- 3-Role Authentication (Owner, Mechanic, Admin)
- Vehicle Management
- Service History Tracking
- Auto Reminder Engine
- Admin Analytics Dashboard
- CSV Export
- PDO Prepared Statements
- UFW Firewall Security
